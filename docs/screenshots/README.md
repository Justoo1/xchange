# Screenshots

The README references three images. Drop your captures here with these exact names:

| File | Screen to capture |
| --- | --- |
| `card.png` | The **My Card** home screen (tab 1) |
| `radar.png` | The **Exchange → Nearby** radar |
| `confirm.png` | The **Confirm & Save** sheet (after an exchange) |

**How to capture (Expo Go on a real device is easiest):**
- Android: press **Power + Volume‑Down**.
- iOS device: **Side button + Volume‑Up**; iOS Simulator: **⌘S**.

Portrait phone screenshots look best. Transfer them to this folder, commit, and push —
they'll render side‑by‑side at the top of the main README.
